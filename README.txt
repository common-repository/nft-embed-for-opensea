Embed any NFT or NFT collection from OpenSea.io into your posts/pages

== Description ==

Opensea integration plugin allows you to embed any NFT anywhere on posts or pages within your website with a simple shortcode. 

You can also embed complete NFT collections.

You can embed NFTs and collections from sites like Etherscan.io or OpenSea.io

== Features ==
* NFT Embed on wordpress posts or pages or anywhere
* Shortcode to embed anywhere
* Buy now button for NFT purchase
* Enable any number of NFTs

== Pro Features ==
* Customize the look and feel of your embed
* Create NFT collection or gallery
* NFT collection embed from OpenSea
* NFT import from OpenSea into woocommerce and other plugins
* NFT ownership validation
* Display user NFTs on user profile
